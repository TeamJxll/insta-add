# INSTA ADD

Automatización de interacciones en Instagram mediante sesiones conectadas.  
Este proyecto permite gestionar cuentas de Instagram y realizar **likes automáticos** de manera controlada, ideal para pruebas de automatización y aprendizaje sobre integración con APIs y sesiones.

---

## 🚀 Características
- Conexión de sesiones de Instagram.
- Automatización de likes en publicaciones.
- Código modular y fácil de adaptar.
- Pensado para desarrolladores que quieran experimentar con bots y automatización.
